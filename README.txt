Moin, das hier wird (vielleicht) ein Programm mit dem der XArm 5 von uFactory ein Bild mit einem Stift an eine Tafel malen kann.

Um den Arm zu bedienen schließt man ihn per LAN an einen Computer an der dieses Programm ausführt, zudem muss die aktuelle Version der Herstellersoftware installiert sein (nähere informationenen evl. in Kürze, angabe ohne Gewähr). Auf disem führt man einmal die Batchdatei "bibliothekeninstallieren.bat" aus, welche die für die GUI notwendigen Bibliotheken installiert. Anschließend GUI wird in Python 3.10 ausgeführt.
